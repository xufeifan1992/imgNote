### 红黑树
<br>
<br>
![enter description here](https://www.github.com/xufeifan1992/note/raw/master/images/2019423/1556009298953.png)

<br>
<br>

* 每个节点或者是红色的，或者是黑色的
* 根节点是黑色的
* 每一个叶子节点(最后的空节点)是黑色的
* 如果一个节点是红色的，那么他的孩子节点都是黑色的
* 从任意一个节点到叶子节点，经过的黑色节点是一样得

<br>
